#  AQL (ArangoDB) language support plugin for Intellij IDEA 2019.2+ IDE


## Build

```
gradle build

```
Note: 
build might take a while because it downloads Intellij Community Edition (300+MB, not sure exactly)


## Run

```
gradle runIde

```
