ASIN()

ASIN(value) → num

Return the arcsine of value.

  • value (number): the input value
  • returns num (number|null): the arcsine of value, or null if value is outside the valid range -1 and 1 (inclusive) 
ASIN(1) // 1.5707963267948966
ASIN(0) // 0
ASIN(-1) // -1.5707963267948966
ASIN(2) // null
